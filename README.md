# comet-pt100

A simple PT100 setup

### Required instruments:

* CTS Climate Chamber
* Keithley 2700 Multimeter/Switch with equipped switch card

## Install

Install using pip in a virtual environment.

```bash
pip install git+https://github.com/hephy-dd/comet-pt100.git
comet-pt100
```
